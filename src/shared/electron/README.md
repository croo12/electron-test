# Electron 통신 레이어

이 디렉토리는 Electron 메인 프로세스와 렌더러 프로세스 간의 통신을 담당하는 코드가 위치합니다.

## 구조

- `index.ts`: Electron 메인 프로세스의 진입점
- (추후) `ipc/`: IPC(Inter-Process Communication) 관련 코드
